How Verification Works
  1. You re-upload the file you want to verify.
  2. The server computes its SHA-256 hash.
  3. The hash is compared with the one stored immutably on the blockchain.
  4. Match → File is authentic and unmodified.
  5. Mismatch → File has been tampered with or replaced.
  6. The verification event is logged on the blockchain automatically.
