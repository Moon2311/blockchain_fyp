What happens after upload?
  1. SHA-256 hash of your file is computed.
  2. File is (optionally) encrypted with AES-256 (Fernet).
  3. Hash + metadata is sent to the ChainOfCustody smart contract on Ganache.
  4. A transaction is mined and the record becomes immutable.
  5. You can later re-upload the file to verify its integrity at any time.
